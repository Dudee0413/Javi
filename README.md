# Kutató Csoport Weboldal

Ez egy egyszerű statikus weboldal egy kutató csoport számára.

## Használat

Nyissa meg az `index.html` fájlt egy webböngészőben.

## Fájlok

- `index.html`: A fő HTML fájl.
- `style.css`: Stíluslap.
- `script.js`: JavaScript fájl sima görgetéshez.

## Hibaelhárítás

Ha a stílusok nem töltődnek be, ellenőrizze, hogy a `style.css` ugyanabban a könyvtárban van-e, mint az `index.html`.# Javi
